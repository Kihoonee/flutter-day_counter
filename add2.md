#개선하고 싶은 기능
## 수정화면
1. 수정화면에서 기준일  항목은 없어도 될거 같아. 목표일만 있어도 돼. 기준일은 항상 오늘이야. 
2. 테마 컬러를 5개 정도 더 추가해줘. 
3. 이벤트 종류 아이콘을 테마처럼 선택 할수 있으면 좋겠어. 10개 정도 추가해줘.
3-1. 지금은 아이콘이 테마와 바인딩되어서 의미없이 나오고 있어. 
4. 삭제버튼 컬러를 좀 넣어줘. 그레이 컬러. 다크모드에서도 구분되는 그레이. 
5. 폰트가 bold로 나오는데 normal로 변경해줘.   할일/다이어리/수정 이벤트 제목과 폰트가 달라. 폰트 통일성있고 세련되고 깔끔하게 잘 정돈되게 해줘. 
6. 당일포함/주말제외 항목을 가장 아래에 배치해줘. 이벤트종류와 테마를 상단에 배치 해줘. 
7. 이벤트 제목 항목이 조금 가독성이 떨어지고 밋밋해. 다른거보다 좀 더 중요한거처럼 보이게 해줘. 

## 다이어리
1. 다이어리 작성해보세요 폰트를 좀 더 작게 해줘. 폰트 컬러 그레이로... textholder 컬러. 
2. 다이어리 아이콘이 너무 안좋아. 다른 아이콘으로 교체해줘. 아이콘이 필요하면 웹에서 서칭해서 카드 디자인과 어울리는 아이콘으로 교체해줘.       
3. 다이어리 작성버튼 원형으로 해줘. 메인화면의 + 버튼과 같은 스타일로 해줘. 위치도 맞춰줘. 
4. 다이어리 작성할때 나오는 달력의 크기가 너무 큰데. 수정화면의 달력과 크기를 맞춰줘. 디자인도 통일해줘. 
5. 다이어리 작성화면 UI 디자인이 맘에 안들어. MyDays 앱의 다이어리 UI를 참고해서 세련되고 깔끔하게 잘 정돈되게 해줘. 
6. 작성된 다이어리 카드의 메뉴버튼은 연필모양으로 수정해줘. 누르면 수정화면으로 이동해줘. 
7. 삭제는 할일 카드 처럼 슬라이드로 삭제되게 수정해줘. 

## 할일
1. 할 일 추가...  폰트 사이즈 조금만 줄이고 폰트 컬러 그레이로... textholder 컬러. 
2. 할 일을 추가해보세요. 폰트 사이즈 조금만 줄이고 폰트 컬러 그레이로... textholder 컬러. 
3. 할일 아이콘 너무 안좋아. 다른 아이콘으로 교체해줘. 아이콘이 필요하면 웹에서 서칭해서 카드 디자인과 어울리는 아이콘으로 교체해줘.

## 메인화면
1. 상단에 Day Counter 앱이름을 Days+ 로 변경해줘.

5. 이벤트 카드안에 카운트숫자 폰트 위치를 조금 아래로 내려줘. 폰트 사이즈 조금 줄여줘. 
6. 이벤트 카드안에 물결웨이브 곡률을 더 과감하게 매우 과감하게 곡률을 줘도 돼. 
6. D-day  + 일경우 - 일경우에 차이를 주고 싶어. 아주 살짝. ( 아이디어 있으면 줘봐 )




#개선하고 싶은 기능 3



## 메인화면
1. 이벤트 카드 드래그앤 드롭으로 순서 변경 할때 픽한 카드의 스케일이 살짝 커지면서 살짝 떠있는 느낌을 주고 싶어. 
2. 이벤트 카드 위아래 간격을 조금 줄여줘. 
3. 이벤트 카드안에 목표날 폰트 위치를 살짝 오른쪽으로 indentation을 조금 더 줘도 돼. 
4. 이벤트 카드안에 카운트숫자 폰트 위치를 조금 아래로 내려줘. 상단 타이틀과 top의 간격과 동일한 간격으로 맞춰줘. 
5. 이벤트 카드안에 카운트 숫자 뒤에 days를 빼줘.  past tag도 빼줘. 
6. 목표날이 미래일때 표기가 - 마이너스 인데, 매우 부정적인 느낌이야. 다른 표기 방법이 있을지 아이디어 부탁해. 



## 수정화면 and 처음입력화면. 
1. '어떤 날인가요?'텍스트 컬러를 다른 항목과 동일하게 그레이로 해줘.  '어떤 날인가요?' 항목 선택하면 나오는 textholder 내용 '이벤트 제목을 입력하세요' 의 폰트크기와 컬러를 조금 조정해줘. 위치도 '어떤 날인가요?' 항목보다 좀더 아래로 낮춰줘. 수정화면 처음입력화면 둘다에 동일하게 적용해줘. 통일된 UI로 맞춰줘.
2. 목표일의 날짜 폰트 컬러도 그레이로 해줘.  목표일 항목의 달력아이콘이 맘에 안들어. 위치도 맘에 안들어.    
3. 아이콘과 테마컬러를 선택하면 카드에 바로 반영되면 좋겠어. 미리보기처럼.    
4. 아이콘과 테마컬러의 정렬을 맞춰줘. 크기도 맞춰줘. 디테일이 중요해.
