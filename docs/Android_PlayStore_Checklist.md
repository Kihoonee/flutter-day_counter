# Days+ 구글 플레이스토어 심사 점검 리포트 (2025)

비즈니스 계정(사업자)으로 구글 플레이스토어에 등록하기 위한 사전 준비 및 기술적 체크리스트입니다.

## 🏢 1. 비즈니스 계정 등록 요건

| 항목 | 상세 내용 | 비고 |
| :--- | :--- | :--- |
| **DUNS 번호** | 국제 사업자 등록 번호 | **필수 (가장 먼저 준비)** |
| **법인 증빙 서류** | 사업자 등록증 등 | 영문 병기 권장 |
| **결제 프로필** | 사업자 명의 카드 | 등록비 $25 |

## 🛠 2. 기술적 요건 (Android 15 대응)

- **Target SDK**: **35 (Android 15)** ✅ (최신 기준 충족)
- **Min SDK**: 24 ✅ (Firebase/AdMob 호환 가능)
- **APK 형식**: 최종 배포 시 `.aab` (App Bundle) 권장

## ⚖️ 3. 주요 정책 준수 사항

1.  **데이터 보안 섹션 (Data Safety)**: Play Console 등록 시 수집 데이터와 광고 ID 사용을 명확히 선언해야 합니다.
2.  **계정 삭제 의무화**: 앱 내에서 계정 탈퇴 및 데이터 삭제 기능을 제공해야 합니다.
3.  **개인정보처리방침**: 항상 최신 가동 중인 웹 링크가 필요합니다.
