#개선하고 싶은 기능
## 수정화면
1. 수정화면에서 기준일  항목은 없어도 될거 같아. 목표일만 있어도 돼. 기준일은 항상 오늘이야. 
2. 테마 컬러를 5개 정도 더 추가해줘. 
3. 이벤트 종류 아이콘을 테마처럼 선택 할수 있으면 좋겠어. 10개 정도 추가해줘.
3-1. 지금은 아이콘이 테마와 바인딩되어서 의미없이 나오고 있어. 
4. 삭제버튼 컬러를 좀 넣어줘. 그레이 컬러. 다크모드에서도 구분되는 그레이. 
5. 폰트가 bold로 나오는데 normal로 변경해줘.   할일/다이어리/수정 이벤트 제목과 폰트가 달라. 폰트 통일성있고 세련되고 깔끔하게 잘 정돈되게 해줘. 
6. 당일포함/주말제외 항목을 가장 아래에 배치해줘. 이벤트종류와 테마를 상단에 배치 해줘. 
7. 이벤트 제목 항목이 조금 가독성이 떨어지고 밋밋해. 다른거보다 좀 더 중요한거처럼 보이게 해줘. 

## 다이어리
1. 다이어리 작성해보세요 폰트를 좀 더 작게 해줘. 폰트 컬러 그레이로... textholder 컬러. 
2. 다이어리 아이콘이 너무 안좋아. 다른 아이콘으로 교체해줘. 아이콘이 필요하면 웹에서 서칭해서 카드 디자인과 어울리는 아이콘으로 교체해줘.       
3. 다이어리 작성버튼 원형으로 해줘. 메인화면의 + 버튼과 같은 스타일로 해줘. 위치도 맞춰줘. 
4. 다이어리 작성할때 나오는 달력의 크기가 너무 큰데. 수정화면의 달력과 크기를 맞춰줘. 디자인도 통일해줘. 
5. 다이어리 작성화면 UI 디자인이 맘에 안들어. MyDays 앱의 다이어리 UI를 참고해서 세련되고 깔끔하게 잘 정돈되게 해줘. 
6. 작성된 다이어리 카드의 메뉴버튼은 연필모양으로 수정해줘. 누르면 수정화면으로 이동해줘. 
7. 삭제는 할일 카드 처럼 슬라이드로 삭제되게 수정해줘.
