# Days+ 출시 폴리싱 및 공유 기능 구현 계획 (Reboot)

안정적인 버전에서 다시 시작하여, 핵심 기능인 '포스터 공유'와 'Apple 심사 대비 설정창 보강'을 구현합니다.

## User Review Required
> [!NOTE]
> 롤백 이전 버전에서 발생했던 렌더링 에러를 방지하기 위해, 디자인 변경은 최소화하고 기능적 완결성에 집중할 예정입니다.

---

## Proposed Changes

### 1. 소셜 공유 기능
- `screenshot`과 `share_plus` 패키지를 사용하여 상세 페이지의 포스터 카드를 이미지로 공유합니다.
- 깔끔한 워터마크("Days+")를 포함한 고해상도 이미지를 생성합니다.

### 2. Apple 심사 대응 (설정 페이지)
- 앱 버전 표시, 개발자 문의하기, 오픈소스 라이선스 확인 기능을 추가합니다.
- Apple App Store 리뷰 가이드라인을 준수하는 UI 요소를 정비합니다.

### 3. '기억의 순간' 안내 UX
- 할일-메모 자동 전환 후 유저에게 그 사실을 알리는 감성적인 팝업을 구현합니다.

---

## Verification Plan
### Automated Tests
- 공유 서비스 호출 로직 검증
- l10n 다국어 키 존재 여부 확인

### Manual Verification
- iOS 시뮬레이터 및 Android 에뮬레이터에서 UI 렌더링 확인
- 실제 공유 시 이미지 품질 및 워터마크 위치 점검
