# Days+ 애니메이션 강화 제안서 (Animation Proposal)

Days+의 **현대적이고 감성적인 파스텔 UI**를 유지하면서, 사용자 경험을 한 단계 높여줄 **심플하고 세련된** 애니메이션 계획을 제안합니다. 과하지 않은 마이크로 인터랙션으로 "살아있는" 앱의 느낌을 주는 것이 핵심입니다.

---

## 🎨 애니메이션 핵심 컨셉: "Soft & Fluid"
- **간결함**: 화려한 효과보다는 자연스러운 움직임에 집중.
- **부드러움**: 급격한 변화 대신 Curves.easeInOut 또는 Curves.elasticOut 사용.
- **일관성**: 모든 화면 전환과 버튼 반응에 동일한 물리 법칙 적용.

---

## 1. 진입 애니메이션 (Staggered Entrance)
메인 화면 진입 시 리스트가 한꺼번에 나타나지 않고, 순차적으로 위로 떠오르며 나타나는 방식입니다.

- **대상**: 메인 화면의 `PosterCard` 리스트.
- **효과**: 하단에서 위로 20px 정도 부드럽게 이동하며 Fade-in (지연 시간 50ms씩 차등 부여).
- **기대 효과**: 앱을 켰을 때 정적인 느낌 대신 리드미컬하고 고급스러운 인상을 줍니다.

## 2. 히어로 전환 (Shared Element Transition)
카드를 클릭했을 때 해당 카드가 상세 화면의 배경으로 자연스럽게 확장되는 방식입니다.

- **대상**: `EventListPage`의 카드 → `EventDetailPage`로의 전환.
- **효과**: Flutter의 `Hero` 위젯을 사용하여 카드 자체가 상세 페이지 전체로 커지는 듯한 연결성 제공.
- **기대 효과**: 화면이 바뀔 때 끊김 없는(seamless) 경험을 제공하여 앱의 완성도를 높입니다.

## 3. 배경 웨이브 마이크로 인터랙션 (Wave Subtle Motion)
`PosterCard` 배경의 Wave 패턴에 아주 미세한 움직임을 부여합니다.

- **대상**: `PosterCard` 내부의 `_PatternPainter`.
- **효과**: 사용자가 카드를 누르고 있거나 화면을 스크롤할 때, 배경의 물결이 아주 천천히 좌우로 출렁이는 효과. (또는 자이로스코프 센서를 이용해 폰을 기울일 때 미세하게 움직임)
- **기대 효과**: "정지된 이미지"가 아닌 "생동감 있는 공간"이라는 느낌을 전달합니다.

## 4. 인터랙티브 피드백 (Interactive Micro-feedback)
사용자의 동작에 즉각적이고 귀여운 반응을 보여줍니다.

- **체크리스트 완성**: 할 일 완료 시 선이 그어지는 애니메이션과 함께 카운트 숫자가 톡 튀어오르는 효과.
- **버튼 탭**: 하단 '기념일 추가' 버튼이나 설정 항목 클릭 시, 살짝 눌렸다가 떼어지는 Scale 애니메이션 (0.95 -> 1.0).
- **진행률 바**: 디데이 진행률이 0%에서 현재 퍼센트까지 부드럽게 채워지는 효과.

---

## 🛠 제안 드리는 구현 순서 (Roadmap)

1.  **Phase 1 (Low Cost, High Impact)**: 리스트 진입 및 버튼 탭 애니메이션 (즉각적인 변화 체감 가능)
2.  **Phase 2 (UX Integration)**: Hero 전환 효과 적용 (상세 페이지와의 연결성 강화)
3.  **Phase 3 (Artistic Polish)**: 배경 웨이브 움직임 및 진행률 필링 (감성적인 완성도 완성)
