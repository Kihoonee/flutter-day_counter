# Competitor Analysis: MyDays - Life Log

사용자 요청에 따라 **MyDays** 앱과 현재 개발 중인 **Day Counter** 앱의 기능을 비교 분석했습니다.

## 📊 기능 비교표 (Feature Comparison)

| 기능 카테고리 | Day Counter (현재 내 앱) | MyDays (벤치마킹 앱) | 분석 / 제안 |
| :--- | :---: | :---: | :--- |
| **홈 화면 위젯** | ❌ (미지원) | ⭕ (소/중/대 지원) | **[추천]** 가장 시급한 추가 기능. iOS/Android 위젯 지원 필요. |
| **디자인/테마** | ⭕ (10종 파스텔, Wave 패턴) | ⭕ (카테고리별 색상) | 내 앱이 더 현대적이고 감성적인 UI (Glassmorphism -> Flat Pastel). |
| **다이어리 (기록)** | ❌ (기능 없음) | ⭕ (날짜별 메모 지원) | 디데이와 연관된 추억 기록 기능 확장 가능성. |
| **할 일 (To-Do)** | ❌ (기능 없음) | ⭕ (준비물 체크리스트) | 디데이(시험, 여행) 준비를 위한 체크리스트 기능. |
| **공유 (Share)** | ❌ (기능 없음) | ⭕ (감성 카드 이미지 공유) | 인스타/카톡 공유를 위한 이미지 생성 기능. |
| **알림 (Notification)** | ❌ (기능 없음) | ⭕ (당일 알림 등) | Firebase Messaging이 준비되어 있으므로 구현 용이. |
| **상단바 고정** | ❌ (기능 없음) | ⭕ (상단바 고정 지원) | 안드로이드 전용 기능. (요즘 트렌드는 아님) |
| **백업/복원** | ❌ (로컬 Only) | ⭕ (오프라인 중심) | 현재 둘 다 로컬 중심. 향후 클라우드 백업 차별화 가능. |

## 💡 추가 제안 기능 (Backlog)

위 비교를 바탕으로 추가 가능한 기능 후보입니다. 원하시는 기능을 선택해 주세요.

1.  **홈 위젯 (Home Ops Widget)**: 앱을 켜지 않고도 홈 화면에서 바로 D-Day 확인 (우선순위 높음)
2.  **공유하기 (Share as Image)**: 현재 예쁜 카드 디자인을 이미지로 저장하거나 카톡으로 공유
3.  **메모/기록 (Memo)**: 각 이벤트마다 간단한 메모 남기기
4.  **알림 (Push Notification)**: D-Day 당일 또는 D-100 등에 푸시 알림 발송

어떤 기능부터 추가할까요? 말씀해 주시면 구현 계획을 세우겠습니다.
