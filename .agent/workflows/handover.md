---
description: Windows와 Mac 사이의 작업 맥락(AI 기억) 동기화 방법
---

낮(Windows)과 밤(Mac) 환경을 오갈 때, 코드뿐만 아니라 AI 에이전트의 작업 맥락(진행 상황, 계획 등)을 완벽하게 이어가기 위한 단계입니다.

## 🏁 세션 종료 시 (Handover 준비)

작업을 마치고 다른 PC로 넘어가기 전, 다음 단계를 수행하세요.

1.  **에이전트에게 맥락 저장 요청**: 
    - 에이전트에게 **"지금까지의 진행 상황을 .antigravity 폴더에 업데이트해줘"**라고 요청합니다. 
    - 에이전트가 `task.md`, `implementation_plan.md`, `walkthrough.md` 등을 프로젝트 내부로 복사합니다.
2.  **Git Push**:
    - 모든 변경 사항과 `.antigravity` 폴더를 커밋하고 푸시합니다.
    ```powershell
    git add .
    git commit -m "chore: handover context for sync"
    git push
    ```

## 🚀 새 세션 시작 시 (Context 복구)

다른 PC에서 작업을 시작할 때 다음 단계를 수행하세요.

1.  **Git Pull**:
    - 최신 코드와 맥락 파이를 가져옵니다.
    ```bash
    git pull
    ```
2.  **에이전트 깨우기**:
    - 에이전트에게 다음과 같이 말합니다:
    > **".antigravity 폴더에 있는 파일들을 읽고 현재 프로젝트 상황과 계획을 파악해줘. 그리고 [오늘의 목표]를 시작하자."**
3.  **에이전트의 인지 확인**:
    - 에이전트가 파일들을 읽고 현재 상태를 요약하면, 이어서 페어 프로그래밍을 진행합니다.

---
> [!TIP]
> 이 워크플로우를 사용하면 서로 다른 OS 환경에서도 AI가 "우리가 어디까지 했지?"라고 묻는 일 없이 바로 동기화됩니다.
