
# 기능개선

## 이벤트 하위 메뉴
1. 할일과 한줄메모의 기능을 분리 해보고 싶어. 
2. d-day, d day 일 경우에는 할일 메뉴가 나오고
3. d+day 일 경우에는 한줄메모가 나오게 변경해줘. 
4. 수정 메뉴는 두 경우 모두 나오도록 해줘. 
5. 탭 메뉴가  할일 | 수정,   한줄메모 | 수정   이렇게 되는거지. 


## 추가 기능
1. d-day, d day  상태일때 작성해놓은 할일의 내용이 d+day 상태로 변경되면 자동으로 할일의 내용을
한줄메모로 기록되면 좋겠어. 할일이 작성된 날짜에 한줄메모에 작성되고 같은날에 할일이 여러개여도 한줄메모에 모두 작성되도록 해줘.
