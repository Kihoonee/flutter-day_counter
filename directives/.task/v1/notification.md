# 알림 기능 명세 (Current)

현재 `Days+` 앱에 적용되어 있는 알림(Push Notification) 로직은 다음과 같습니다.

## 1. 기본 설정
- **알림 시간**: 모든 알림은 해당 날짜의 **오전 9:00**에 발송됩니다.
- **기술 스택**: `flutter_local_notifications`, `timezone` 패키지를 사용한 로컬 푸시 알림.

## 2. 이벤트 상태별 알림 로직

### A. 미래 이벤트 (D-Day 전)
- **D-1 알림**: 목표일 1일 전 오전 9시에 발송.
    - 제목: `{이벤트명} D-1`
    - 내용: `하루 남았습니다! 준비되셨나요?`
- **D-Day 알림**: 목표일 당일 오전 9시에 발송.
    - 제목: `{이벤트명} D-Day!`
    - 내용: `오늘은 {이벤트명} 날입니다.`

### B. 과거 이벤트 (D-Day 후)
- **기념일(Milestone) 알림**: 기념일로부터 **100일 단위**(+100일, +200일, ...)로 자동 스케줄링.
    - 제목: `{이벤트명} +{N}일`
    - 내용: `시작한 지 {N}일이 되었습니다.`

## 3. 알림 관리
- **이벤트 업데이트/삭제 시**: 해당 이벤트와 관련된 모든 예약된 알림(D-Day, D-1, 기념일)을 즉시 취소하고 재설정합니다.
- **ID 생성**: 이벤트 ID와 알림 타입(dday, dminus1, anniv)을 조합한 해시값을 ID로 사용하여 충돌을 방지합니다.
