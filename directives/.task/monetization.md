# Days+ 수익화 전략 문서

## 1. 현황

- **적용 완료**: 메인 화면(EventListPage) 하단 배너 광고
- **AdMob SDK**: `google_mobile_ads` 패키지 사용
- **앱 특성**: D-Day 카운터 유틸리티 앱 (짧은 세션, 잦은 재방문)

---

## 2. 광고 배치 전략

### 2.1 권장 광고 유형별 배치

| 우선순위 | 광고 유형 | 배치 위치 | 예상 수익 기여도 | UX 영향 |
|:---:|:---|:---|:---:|:---:|
| 1 | 배너 광고 | 메인 화면 하단 | ★★☆ | 낮음 ✅ |
| 2 | 앱 오픈 광고 | 앱 시작/백그라운드 복귀 | ★★★ | 중간 |
| 3 | 전면 광고 | 이벤트 상세→목록 복귀 | ★★★ | 높음 ⚠️ |
| 4 | 네이티브 광고 | 이벤트 목록 사이 | ★★☆ | 낮음 ✅ |

### 2.2 상세 구현 계획

#### ✅ 배너 광고 (적용 완료)
- **위치**: `EventListPage` 하단 SafeArea 영역
- **크기**: Adaptive Banner (화면 너비에 맞게 자동 조절)

#### 🎯 앱 오픈 광고 (1순위 추가)
```
트리거 조건:
- 앱 최초 실행 시
- 백그라운드에서 포그라운드 복귀 시 (3분+ 경과)

제한 조건:
- 24시간 내 최대 5회
- 연속 표시 간격: 최소 60분
```

#### 🎯 전면 광고 (2순위 추가)
```
트리거 조건:
- 이벤트 상세 화면에서 뒤로가기 시

빈도 제한:
- 5회 화면 전환당 1회 표시
- 세션당 최대 2회
- 이벤트 생성/수정 직후에는 표시하지 않음
```

#### 📋 네이티브 광고 (선택)
```
배치 위치:
- 이벤트 목록 5번째 카드 이후 (매 7개마다 1개)

조건:
- 이벤트가 3개 이상일 때만 표시
```

## 3. 사용자 저항감 최소화 전략

### 3.1 핵심 원칙

> **"광고는 사용자의 핵심 작업을 방해하지 않아야 한다"**

| 절대 금지 | 권장 |
|:---|:---|
| ❌ 이벤트 추가/수정 중 광고 | ✅ 작업 완료 후 자연스러운 전환점에서 |
| ❌ 알림 확인 직후 광고 | ✅ 앱 시작 시 (콘텐츠 보기 전) |
| ❌ 5초 미만 간격으로 광고 | ✅ 충분한 간격 유지 |

### 3.2 스마트 빈도 제어

```dart
/// 광고 표시 조건 체크
bool shouldShowAd() {
  // 1. 마지막 광고 후 최소 60초 경과
  // 2. 세션당 최대 표시 횟수 미달
  // 3. 핵심 작업(생성/수정) 중이 아님
  // 4. 첫 실행 후 최소 3개 이벤트 생성 완료
}
```

### 3.3 첫 사용자 경험 보호 (온보딩 기간)

| 기간 | 광고 정책 |
|:---|:---|
| 앱 설치 후 24시간 | 배너만 표시 |
| 이벤트 3개 생성 전 | 전면/앱오픈 광고 비활성화 |
| 첫 주간 | 전면 광고 빈도 50% 감소 |

## 4. 수익 극대화 방안

### 4.1 광고 유형별 기대 수익 (참고치)

| 광고 유형 | eCPM 범위 (USD) | 노출 빈도 | 월간 예상 수익* |
|:---|:---|:---|:---|
| 배너 | $0.5 ~ $2 | 상시 | $50 ~ $200 |
| 앱 오픈 | $5 ~ $15 | 일 3~5회 | $75 ~ $225 |
| 전면 | $3 ~ $10 | 세션당 1~2회 | $45 ~ $150 |
| **합계** | | | **$170 ~ $575** |

*DAU 1,000명 기준 예상치

### 4.2 A/B 테스트 권장 항목

1. **전면 광고 빈도**: 5회 vs 7회 vs 10회당 1회
2. **앱 오픈 광고 최소 이탈 시간**: 1분 vs 3분 vs 5분
3. **네이티브 광고 위치**: 3번째 vs 5번째 카드 후

### 4.3 프리미엄(광고 제거) 옵션

```
💎 Days+ Pro
- 모든 광고 제거
- 가격: ₩4,900/영구 또는 ₩1,900/월
- 수익화 보완: 광고 싫어하는 사용자도 수익화
```

## 5. 구현 우선순위

- [ ] **Phase 1**: 앱 오픈 광고 추가 (1~2일)
- [ ] **Phase 2**: 전면 광고 + 빈도 제어 로직 (2~3일)  
- [ ] **Phase 3**: 프리미엄(광고 제거) 인앱 결제 (3~5일)
- [ ] **Phase 4**: 네이티브 광고 (선택, 4~5일)
